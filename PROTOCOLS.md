Ai research:
Yes, several Solana protocols operate with a modular "primitive" architecture similar to Morpho Blue, allowing you to create isolated lending markets for any asset without needing to be whitelisted by the protocol team.

1. Kamino Lend V2

Kamino Finance's Lend V2 is the closest equivalent to Morpho Blue on Solana.


Permissionless Creation: Anyone can use the Kamino Manager SDK to deploy a "Market Layer" with any combination of assets.
Modular Architecture: It separates the Market Layer (where raw lending occurs) from the Vault Layer (where curators like Re7 Labs or Steakhouse Financial manage risk). This mirrors Morpho Blue and MetaMorpho.
Customization: You define the oracle, interest rate curve, and Loan-to-Value (LTV) ratios yourself. 


2. Save (formerly Solend)

Save Finance (rebranded from Solend) pioneered permissionless pools on Solana.


Creation Fee: You can spin up an isolated pool for any SPL token by paying a flat fee of 100 SLND (approx. $60–$70).
Incentives: As the creator, you automatically receive 20% of the borrow fees generated by that specific pool.
No Whitelist: There is no manual approval process; if you have the fee and the assets, you can launch the pool instantly. 


3. Jupiter Offerbook

Launched in early 2026, Jupiter Offerbook provides a permissionless P2P credit layer.


Asset Support: It is designed for "every asset on Solana," including long-tail tokens, NFTs, and Real-World Assets (RWAs).
Mechanism: Unlike the pool-based models of Kamino or Save, Offerbook uses time-based P2P loans without price-based liquidations, allowing for more flexible, direct matching between users. 


4. Marginfi (The Arena)

Marginfi offers a dedicated section called The Arena for experimental and long-tail assets.


Isolated Banks: Creating a "bank" for a new token is permissionless, and the creator acts as the admin for that specific group.
Vetting Note: While bank creation is permissionless, the UI often flags or separates these from the main "mrgnlend" pools to protect users from unverified assets.
